# Stage-Manager-Gnome
A dynamic GNOME Shell sidebar that brings a Stage Manager–like workflow to Linux.

Just moved to linux this and my workflow was terrible without stage manager. So using chat gpt tried to creat this extension as an alternative way to deal with too many windows on small screens. Its just a clone of stage manager.

It is especially useful on laptops or compact displays.

# Installation

Open the terminal and run:

1- git clone https://github.com/magoness/Stage-Manager-Gnome.git

2- mv Stage-Manager-Gnome ~/.local/share/gnome-shell/extensions/stagemode@mago

3- gnome-extensions enable stagemode@mago

Log out and log back in.
