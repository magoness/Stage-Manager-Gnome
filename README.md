# Stage-Manager-Gnome
A dynamic GNOME Shell sidebar that brings a Stage Manager–like workflow to Linux.

Just moved to linux this yaer and my workflow was terrible without stage manager so using chat gpt tried to creat this extension as an alternative way to deal with too many open windows on small screens. So its just a clone of stage manager.

It is especially useful on laptops or compact displays where space is limited.

# Installation

1- Download the ZIP from GitHub.
2- Extract it.
3- Move the folder to: ~/.local/share/gnome-shell/extensions/
